## 推奨画像仕様

- 形式: JPG または PNG
- サイズ: 400x400px 以上
- アスペクト比: 1:1 (正方形) 推奨
- ファイルサイズ: 500KB以下推奨

## 注意事項

画像が存在しない場合は、自動的にプレースホルダー画像が表示されます。
実際の商品画像を配置することで、より魅力的なECサイトになります。
